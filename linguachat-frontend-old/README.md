# LinguaChat Frontend

React frontend for LinguaChat.

The interface includes CEFR level selection, dark mode, animated loading,
structured correction cards, and guided-practice prompts.

## Scripts

```bash
npm install
npm start
npm run build
```

Set `REACT_APP_API_URL` in `.env` if the backend is not running at
`http://127.0.0.1:8000`.
